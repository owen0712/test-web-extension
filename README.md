# HTML to Text Saver (Chrome Extension)

This extension lets you save the current page's full HTML source into a `.txt` file.

## Load locally in Chrome

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select this project folder

## Use

1. Open any `http` or `https` page.
2. Click the extension icon.
3. Click **Save HTML as TXT**.
4. Choose where to save the text file.

## Notes

- Chrome internal pages (`chrome://...`) and some restricted pages cannot be scripted.
- File is saved via Chrome's Downloads API.
